# Zuri

> **Your business, online. Beautifully.**
> AI presence activation for African entrepreneurs. Powered by Gemini.

Zuri takes a business owner from zero online presence to a live premium
website, a personalized 90-day content action plan, and access to vetted
execution partners — all starting from a single conversation.

Built for the **XPRIZE Build with Gemini 2026** competition
(Professional Services category).

---

## The Four Pillars

| Pillar | What it does |
|--------|--------------|
| 🌐 **Website Generator** | Gemini composes a premium website from a conversation, live in under 10 minutes |
| ✍️ **Content Studio** | AI drafts posts, newsletters, and animated videos |
| 📅 **Action Plan** | 90-day day-by-day tasks with streaks, badges, and accountability |
| 🤝 **Agency Marketplace** | Matches owners to vetted execution partners |

---

## Tech Stack

- **Framework:** Next.js 15 (App Router, RSC)
- **Styling:** Tailwind CSS + shadcn/ui + Framer Motion
- **Backend:** Supabase (Postgres, Auth, Storage, RLS)
- **AI:** Google Gemini 2.0 Flash + 2.5 Pro
- **Payments:** Flutterwave (NGN-first)
- **Email:** Resend
- **Media:** Unsplash, Pexels, Canva deep-links
- **Hosting:** Vercel (with wildcard subdomains for published sites)
